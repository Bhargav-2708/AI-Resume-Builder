# AI Resume Builder

Premium AI-powered resume builder with real-time editing, professional templates, and AI CV review.

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/Bhargav-2708/AI-Resume-Builder)

## Features
- 🤖 **AI CV Review**: Get professional feedback on your resume using Google Gemini AI.
- 📄 **6 Premium Templates**: Modern, Professional, Creative, and more.
- ⚡ **Real-time Preview**: See changes instantly as you type.
- 🔒 **Secure Auth**: Private account to manage multiple resumes.
- 📥 **PDF Export**: Download high-quality PDFs ready for applications.

## Deployment
Click the button above to deploy this project to Render for free.

### Required Environment Variables
- `GEMINI_API_KEY`: Your Google Gemini API Key.
